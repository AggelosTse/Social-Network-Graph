# Social Network Graph (Python)

This project implements a **Social Network Graph** using graph theory principles in Python.

# Overview

It simulates how social media platforms manage relationships such as followers and friendships by representing:

- Users as **nodes**
- Relationships as **edges**

---

# Features

-  Add a new user (node)
-  Remove an existing user
-  Add relationships between users
-  View neighbors (connections) of a user
-  Display the entire graph structure
-  Save the graph data to a `.txt` file

---

# Functionality

The graph is internally represented using an adjacency list structure for efficient storage and modification of relationships.

This allows:

- Fast lookup of connections
- Easy addition/removal of nodes
- Flexible scaling for larger networks

---
